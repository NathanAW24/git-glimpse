Description:
Closes #798
📝 Description
Update collection in TableContext when items change to update dependent components like Table.Pagination.
⛳️ Current behavior (updates)
Currently the collection are only initialized once, so even if you add items the Pagination won't get additional pages. Only Table.Body uses the collections from props.
🚀 New behavior
When items prop of Table.Body changes, TableContext is updated and Pagination will update the number of pages accordingly.
💣 Is this a breaking change (Yes/No):
No
